# 🧠 SOUL FOUNDER QA: THE AUDIT SYSTEM (SINGLE SOURCE OF TRUTH)

Before generating ANY final code or text, you MUST run your proposed content through this rigorous quality control sequence. 

## 1. USA TONE & HUMAN LANGUAGE AUDIT
Review all written content for tone and authenticity. Ensure that:
- Language sounds natural and human. Tone fits a high-end U.S.-based business.
- No robotic AI phrasing exists. (BANNED: "Delve into", "A symphony of", "Elevate", "Unleash").
- **H1 ASPIRATIONAL CHECK:** Are the H1s and Hero titles aspirational and commercial? If an H1 says the literal service name (e.g., "Roofing Services"), REWRITE IT to be an emotional/powerful hook.

## 2. PAGE-BY-PAGE CONTENT & ASSET ACCURACY
Check each page individually. Confirm that:
- **NO EMOJIS:** Are there any emojis? If yes, DELETE them and use `lucide-react` or SVG icons.
- **IMAGE DENSITY:** Does EVERY page have images/placeholders? A page without images is an instant failure.
- Page content matches the intent. No placeholder text (Lorem Ipsum) remains.

## 3. FOUNDER SCRUTINY REVIEW (FINAL PROMPT)
Act as the founder reviewing this website before client delivery. Ask yourself:
- Would this website pass a high-end U.S.-based agency standard?
- Is anything rushed, sloppy, inconsistent, or unfinished?
- Does all written content sound natural and professional in U.S. English?
- Are all services listed truly relevant and logical for this industry, location, and client type?
- **DESIGN CHECK:** Are the Reviews and Locations modules completely unconventional (no basic maps/carousels)? Is Mobile strictly 1-column?
- **FOOTER CHECK:** Does the footer say exactly "Designed by whitecode.ai"?

List ANY remaining concerns. If something feels “off,” flag it and fix it immediately.

## 4. DELIVERY STANDARD
A website is NOT ready until all tiers are completed, all issues fixed, content is U.S.-appropriate, and the founder-level review passes. This document is the single source of truth.
